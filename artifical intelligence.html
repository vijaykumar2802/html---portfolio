Artificial Intelligence Course - SkillDunia (in collaboration with E-Cell IIT Hyderabad)

October - November 2023

I completed an 8-week intensive Artificial Intelligence course designed to build foundational and practical skills in AI and Machine Learning. The curriculum covered core topics including Python for AI, supervised and unsupervised learning, deep learning fundamentals, and hands-on model development using libraries like Pandas, NumPy, Matplotlib, and Keras.

Key topics and skills:

  • Introduction to AI & ML: Understanding how machines simulate human intelligence, the history of AI, and its real-world applications across industries.
  • Python for AI: Using Python and Jupyter notebooks to manipulate data and build models.
  • Machine Learning Concepts: Supervised vs. unsupervised learning techniques and model training using decision trees.
  • Deep Learning & CNNs: Introduction to deep learning architectures and convolutional neural networks with Keras.
  • Hands-on Projects: Worked on real datasets (e.g., the Iris dataset), implemented classifiers, evaluated models using metrics like accuracy and confusion matrices, and visualized decision trees.
  • Tools & Platforms: Gained practical experience with Google Colab, GitHub for version control, and collaborative coding.

This course strengthened my understanding of AI's technical and conceptual aspects and gave me the opportunity to apply theory through projects and coding labs.
